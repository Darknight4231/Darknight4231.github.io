void ctx.bezierCurveTo(cp1x, cp1y, cp2x, cp2y, x, y); Parameters |||| cp1x The x-axis coordinate of the first control point. cp1y The y-axis coordinate of the first control point. cp2x The x-axis coordinate of the second control point. cp2y The y-axis coordinate of the second control point. x The x-axis coordinate of the end point. y The y-axis coordinate of the end point. Example void ctx.moveTo(); void ctx.bezierCurveTo(); Where 'ctx.moveTo();' defines the start point, not the first control point.
